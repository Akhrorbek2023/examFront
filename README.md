# Movies Frontend (React + Vite + Tailwind)

## Setup

- Copy `frontend/.env.example` to `frontend/.env` (or set `VITE_API_URL`)
- Install deps: `npm install` (run inside `frontend/`)
- Start: `npm run dev`

## Routes

- `/login`
- `/register`
- `/movies`
- `/movies/:id`
- `/movies/new` (protected)
